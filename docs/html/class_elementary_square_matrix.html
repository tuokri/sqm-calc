std::vector<std::shared_ptr<T> > ElementarySquareMatrix< T >::block ( unsigned int  start,
unsigned int  step 
) const
inline

Return a block of pointers from the matrix.

Parameters
startIndex of first element.
stepHow many elements to include.
Returns
Vector of Element pointers.

Function considers indices as if the multi-dimensional matrix was unrolled to a single-dimensional vector.

*  Example matrix:  |   Is considered as:
*  [[2,3,4]         |
*   [1,8,9]         |   [2,3,4,1,8,9,0,0,1]
*   [0,0,1]]        |
*

*

  • Index 0 refers to first element at (0),(0) which is 2.
  • Index 5 refers to sixth element at (1),(2) which is 9.
